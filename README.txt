============================================================
        REEL SENSE AI - Video Ad Generator
        Quick Start Guide (Windows)
============================================================

STEP 1: INSTALL PYTHON (if not installed)
-----------------------------------------
1. Go to: https://python.org/downloads
2. Download Python 3.11 or 3.10
3. Run the installer
4. IMPORTANT: Check the box "Add Python to PATH"
5. Click Install


STEP 2: GET YOUR FREE API KEYS
------------------------------
You need two FREE API keys (takes 2 minutes):

1. GROQ API Key (for script writing):
   - Go to: https://console.groq.com
   - Sign up with Google/GitHub
   - Click "API Keys" > "Create API Key"
   - Copy the key

2. REPLICATE API Token (for video generation):
   - Go to: https://replicate.com
   - Sign up with Google/GitHub
   - Click your profile > "API tokens"
   - Copy the token


STEP 3: RUN THE APP
-------------------
1. Double-click "START.bat"
2. First time: Enter your API keys when asked
3. Browser opens automatically
4. Login with:
   - Email: any email (e.g., test@test.com)
   - Beta Code: REELSENSE2025


STEP 4: CREATE YOUR VIDEO
-------------------------
1. Enter your product/topic (e.g., "Nike running shoes")
2. Choose video quality:
   - 480p = Fast (~1 min per clip)
   - 720p = Medium (~2 min per clip)
   - 1080p = Best quality (~4 min per clip)
3. Click "Generate" and wait
4. Download your video!


TROUBLESHOOTING
---------------
- "Python not found": Install Python from python.org
- Server won't start: Close other programs using port 5000
- Video fails: Check your Replicate account has credits


NEED HELP?
----------
Contact support with any questions.
